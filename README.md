# 🏥 Hospital Management System

A full-stack hospital management application with a React frontend and Spring Boot backend.  
This project helps manage patients, doctors, appointments, and overall hospital operations efficiently.

---
## ✨ Features

- ➕ Add, update, and delete patients  
- 📅 Appointment scheduling  
- 🔗 REST API integration  
- 🌐 Fully responsive UI

## ☁️ Deployment on Azure

The frontend is deployed on **Microsoft Azure** using Docker containers.

### 🔹 Services Used

- Azure App Service (Linux)  
- Azure Container Registry (ACR)  
- Docker  

---
### 🔹 Deployment Workflow

1. Built Docker image for React frontend  
2. Tagged image for Azure Container Registry  
3. Pushed image to ACR  
4. Created Azure Web App (Linux)  
5. Deployed container from ACR  
6. Restarted App Service  

---
